# Little Lemon Table Booking System

A React-based table booking application for Little Lemon restaurant.

## Features
- Date and time selection
- Number of guests picker
- Occasion dropdown (Birthday, Anniversary, Business, etc.)
- Special requests text area
- Form validation
- Booking confirmation screen
- Responsive design

## Technologies Used
- React.js
- JavaScript (ES6+)
- CSS3
- Formik & Yup (form validation)
